📋 Appwrite Console Setup Guide

🗄️ STEP 1: Create Database

  1. Go to: https://cloud.appwrite.io
  2. Select your "TrustPay" project
  3. Click "Databases" in left sidebar
  4. Click "Create Database"
  5. Database ID: trustpay_main
  6. Name: TrustPay Main Database
  7. Click "Create"

📁 STEP 2: Create Collections

Create these 7 collections in your database:

  1. Users - ID: users
  2. Escrows - ID: escrows
  3. Transactions - ID: transactions
  4. Disputes - ID: disputes
  5. KYC - ID: kyc
  6. Notifications - ID: notifications
  7. Invites - ID: invites

📦 STEP 3: Create Storage Buckets

  1. Click "Storage" in left sidebar
  2. Click "Create Bucket" 3 times:
    • Bucket ID: kyc_documents - Name: KYC Documents
    • Bucket ID: proof_files - Name: Proof Files
    • Bucket ID: profile_images - Name: Profile Images
  3. Set permissions: Read/Write/Delete = Any

🔗 STEP 4: Enable Authentication

  1. Click "Auth" in left sidebar
  2. Enable "Email/Password" authentication
  3. Set your domain in allowed domains

🔧 STEP 5: Get Your Credentials

  1. Click "Settings" in left sidebar
  2. Copy your Project ID: 69c165c7003cae4214fe
  3. Copy your Endpoint: https://fra.cloud.appwrite.io/v1
