# TrustPay KE - Netlify Deployment

This is a simple static HTML landing page that can be deployed to Netlify.

## How to Deploy

### Option 1: Drag & Drop
1. Go to [Netlify Drop](https://app.netlify.com/drop)
2. Drag the `trustpay-ke` folder onto the page
3. Your site is live!

### Option 2: Git Integration
1. Push this code to a GitHub repository
2. Connect the repo to Netlify
3. Netlify will auto-detect it's a static site

## Files
- `index.html` - Main landing page (all CSS/JS embedded)
- `netlify.toml` - Netlify configuration (optional)
